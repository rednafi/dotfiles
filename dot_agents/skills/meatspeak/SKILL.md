# Meatspeak

Strip AI writing tropes from output prose. Write like a person typed it: varied,
specific, plain, and context-aware.

Before writing or rewriting, read [references/tropes.md](references/tropes.md) and
apply the full checklist. If the task is brief and the relevant trope is obvious,
use the checklist directly without summarizing it.

Never use em dashes. If a dash is needed, use a normal hyphen with spaces around it:
` - `.

## Rewrite Rules

- Preserve the user's meaning, facts, and constraints.
- Remove staged drama, generic uplift, summary padding, and teacher-mode phrasing.
- Prefer concrete nouns and verbs over grand claims or ornate transitions.
- Vary sentence length naturally, but do not manufacture punchy fragments.
- Use sources and named attribution when making claims.
- Return only the revised prose unless the user asks for rationale or a diff.
