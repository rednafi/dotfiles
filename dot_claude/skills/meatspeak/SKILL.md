# Meatspeak

Strip all AI writing tropes from output prose. Write like a real person typed it - varied,
imperfect, specific. Follow these rules religiously. No exceptions.

Never use em dashes anywhere. If a dash is needed, use a normal hyphen flanked by
spaces: " - ".

The full trope reference is in [references/tropes.md](references/tropes.md). Read it and
follow every rule in it. No exceptions.

## What to do instead

Write plainly. Vary sentence length and structure naturally. Be specific rather than
grandiose. If a simple word works, use it. Let arguments stand on evidence, not rhetorical
tricks. Match the tone to the audience - don't default to teacher mode. When in doubt,
shorter and plainer beats longer and fancier.
